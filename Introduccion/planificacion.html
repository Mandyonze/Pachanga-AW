  • Tareas a realizar

    • Parte gráfica 1

      • Diseño en HTML y CSS de las páginas de búsqueda, inicio de sesión, creación de usaurio, perfil de usuario propio y perfil de otro usuario parta añadir comentarios.

    • Parte gráfica 2

      • Diseño en HTML y CSS de las páginas de creación de partidos, mis partidos, datos del partido, lista de partidos, resultado del partido y valoración de los jugadores.

    • Parte gráfica 3

      • Diseño en HTML y CSS de las páginas de creación de equipos, mejores jugadores en tu zona y aceptar jugador en tu equipo.

    • Parte BD 1

      • Diseño de las bases de datos relacionadas con los usuarios, equipos y los deportes.

    • Parte BD 2

      • Diseño de las bases de datos relacionadas con los eventos.

    • Conexión de base de la base de datos con la interfaz

      • Relacionar las bases de datos con la interfaz de usuario creada en los primeros puntos.
